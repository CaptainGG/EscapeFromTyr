# Pixel Art Direction Skill

Define a production-usable visual language for a pixel-art browser game.

## Deliverables
Write or update:
- `docs/art-direction.md`
- asset requirements in `docs/GDD.md`

## Output
Include:
1. Visual premise
2. Reference style keywords
3. Native resolution assumptions
4. Tile size
5. Sprite scale and proportions
6. Palette strategy
7. Environment style
8. Character style
9. FX style
10. UI iconography style
11. Animation guidance
12. Asset list for the next milestone

## Rules
Prioritize readability, clear silhouettes, practical animation counts, and milestone-sized asset volume. Distinguish hazards, pickups, objectives, and interactables at a glance.
