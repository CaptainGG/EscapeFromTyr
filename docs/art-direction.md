# Station Escape Art Direction

## Visual Premise
Station Escape should feel like a damaged orbital transit hub: dark metal corridors, warning lights, dead signage, and sparse functional glow. The scene should read as sci-fi infrastructure first and dungeon second.

## Reference Keywords
- orbital transit hub
- emergency strip lighting
- industrial sci-fi corridors
- cold metal panels
- readable pixel silhouettes

## Native Resolution
Design for a native playfield of `480x270` inside a responsive browser shell.

## Tile Size
Use a 16x16 grid for wall rhythm, door framing, and room composition.

## Sprite Scale And Proportions
- Player placeholder: 16x16 body with a bright visor.
- Security drone: 16x16 hovering unit with a red optic.
- Bullet: 4x4 bright pulse.
- Keycard: 12x8 flat access tag.
- Exit door: 28x40 frame with locked and unlocked states.

## Palette Strategy
Use deep blues, charcoals, and muted industrial steel with a few high-contrast accents:
- Space black: `#090b14`
- Station floor: `#1a2233`
- Wall steel: `#33435c`
- Screen cyan: `#5edbff`
- Player amber: `#ffc857`
- Drone red: `#ff6b6b`
- Exit mint: `#90f4b2`
- Keycard violet: `#b59cff`

## Environment Style
Favor corridors, pressure doors, floor striping, vent panels, and sparse consoles. Each room should have one obvious silhouette landmark so navigation stays readable.

## Character Style
Keep shapes blocky and functional. The player should look exposed and mobile. Drones should read as blunt automated threats.

## FX Style
Use small muzzle pulses, hit flashes, cooldown glows, and exit power sweeps. Avoid large screen-filling effects.

## UI Iconography Style
Use hard-edged bars, compact labels, and utilitarian framing like a station diagnostic overlay.

## Animation Guidance
- Player: optional movement lean through rotation or recoil.
- Drone: steady hover in patrol, sharper motion in chase.
- Bullet: short, bright travel streak.
- Exit: red locked strip, then green power sweep on unlock.
